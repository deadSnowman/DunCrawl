# DunCrawl

A Dungeon Crawler game in development for the purpose of learning OpenGL
Lots of the code is influenced by thebennybox's tutorials

Currently there is:
* 3D Map generation from bitmap images
* Enemy camera rotation
* Player Movement

To do:
